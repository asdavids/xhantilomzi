# Xhantilomzi — Official Website

Official artist website for **Xhantilomzi** (*Umoya Wami*, 2025).

## Stack

Static single-file HTML/CSS/JS — no framework, no dependencies.

## Pages

- **Home** — Hero, album feature, press quote
- **Music** — Full tracklist with interactive player
- **About** — Bio, influences, live shows, sound breakdown
- **Contact** — Booking, press, press kit request

## Deployment

Hosted on Vercel. Auto-deploys on every push to `main`.
